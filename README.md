# tty-prompt-example

## Setup

```sh
bundle install
rake db:setup
```

## Start

```sh
bundle exec ruby application.rb
```

## Using instruction

### Seeds

- One user (like administrator). Its data for login: `admin`-`password`
- Coins:
  - 10 quarters
  - 10 halfs
  - 10 ones
  - 10 twos
  - 10 threes
  - 10 fives
- Products:
  - 20 pieces of Water by 2.5
  - 2 pieces of Cola by 15
  - 0 pieces of Mars by 10.25
  - 4 pieces of Snickers by 15
